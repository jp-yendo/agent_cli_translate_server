# テーブル定義

設定はデータベースではなく、設定ファイル `~/.agent_cli_translate_server/settings.json` に JSON 形式で保存する。
本書ではファイル内の各セクションを1テーブルとして定義する。

データモデルの実装は `src/shared/models/` 配下に1テーブル1ファイルで配置する。

## common (共通設定)

実装ファイル: `src/shared/models/common-settings.ts`

| カラム名 | 型 | 必須 | デフォルト | 説明 |
| --- | --- | --- | --- | --- |
| host | string | ○ | `127.0.0.1` | 待ち受けアドレス |
| port | number | ○ | `4660` | 待ち受けポート |
| fallbackFrom | string | ○ | `en` | from パラメータ省略時の翻訳元言語コード |
| fallbackTo | string | ○ | `ja` | to パラメータ省略時の翻訳先言語コード |
| agentRetentionSec | number | ○ | `300` | エージェントプロセスの最大稼働時間 (秒)。既存設定との互換性のためキー名は維持 |
| hintId | string \| null | ○ | `null` | 全エンジン共通で利用する翻訳ヒントのID。hints.id への参照。参照先が削除された場合は null に戻す |

## agents (Agent CLI 設定)

実装ファイル: `src/shared/models/agent-cli-config.ts`

Agent CLI の ID (`claude-code` / `codex` / `grok` / `opencode` / `opencode-ollama`) をキーとするマップ。

| カラム名 | 型 | 必須 | デフォルト | 説明 |
| --- | --- | --- | --- | --- |
| maxConcurrency | number | ○ | claude-code/codex/grok: `5`、opencode/opencode-ollama: `1` | 同時起動できるエージェント数 |
| maxUses | number | ○ | `50` | 1プロセスを再作成するまでの最大利用回数 |
| modelName | string \| null | ○ | `null` | OpenCode (Ollama) へ渡す Ollama モデル名。他の Agent CLI では null |

## apiProviders (API プロバイダー設定)

実装ファイル: `src/shared/models/api-provider-config.ts`

API プロバイダーの ID (`ollama` / `openai-compatible` / `anthropic-compatible`) をキーとするマップ。
Agent CLI (ローカルプロセス起動) とは異なり、公式ライブラリ経由でリモート/ローカルの API エンドポイントへ直接リクエストする。

| カラム名 | 型 | 必須 | デフォルト | 説明 |
| --- | --- | --- | --- | --- |
| baseUrl | string | ○ | `''` | エンドポイントの Base URL。空文字の場合はプロバイダー既定値 (ollama: `http://127.0.0.1:11434`、openai-compatible: `https://api.openai.com/v1`、anthropic-compatible: `https://api.anthropic.com`) を使用する |
| apiKey | string | ○ | `''` | API Key。認証不要なローカルサーバー等では空文字 |
| model | string | ○ | `''` | 利用するモデル名。サーバー開始には非空が必須 |
| maxConcurrency | number | ○ | `1` | 同時接続数 (並行して呼び出せる API リクエスト数の上限)。超過分は待ち行列で処理する |

## hints (翻訳ヒント)

実装ファイル: `src/shared/models/translation-hint.ts`

| カラム名 | 型 | 必須 | デフォルト | 説明 |
| --- | --- | --- | --- | --- |
| id | string (UUID) | ○ | 自動生成 | 一意なID。common.hintId から参照される |
| name | string | ○ | - | 表示名 |
| summary | string | ○ | `''` | アプリ概要。翻訳プロンプトの app_context として渡される |

## ui (UI 設定)

実装ファイル: `src/shared/models/ui-settings.ts`

| カラム名 | 型 | 必須 | デフォルト | 説明 |
| --- | --- | --- | --- | --- |
| theme | string | ○ | 初回起動時に OS 設定から判定して保存 | テーマ (`light` / `dark`)。ファイル上の初期値は `system` で、初回起動時に確定値へ置き換わる |
| language | string \| null | ○ | 初回起動時に OS ロケールから判定して保存 | 表示言語 (`ja` / `en`)。ファイル上の初期値は `null` で、初回起動時に確定値へ置き換わる |

## 設定ファイル例

```json
{
  "common": {
    "host": "127.0.0.1",
    "port": 4660,
    "fallbackFrom": "en",
    "fallbackTo": "ja",
    "agentRetentionSec": 300,
    "hintId": "5f7d..."
  },
  "agents": {
    "claude-code": { "maxConcurrency": 5, "maxUses": 50, "modelName": null },
    "codex": { "maxConcurrency": 5, "maxUses": 50, "modelName": null },
    "grok": { "maxConcurrency": 5, "maxUses": 50, "modelName": null },
    "opencode": { "maxConcurrency": 1, "maxUses": 50, "modelName": null },
    "opencode-ollama": { "maxConcurrency": 1, "maxUses": 50, "modelName": "qwen3-coder" }
  },
  "apiProviders": {
    "ollama": { "baseUrl": "", "apiKey": "", "model": "", "maxConcurrency": 1 },
    "openai-compatible": { "baseUrl": "", "apiKey": "sk-...", "model": "", "maxConcurrency": 1 },
    "anthropic-compatible": { "baseUrl": "", "apiKey": "sk-ant-...", "model": "", "maxConcurrency": 1 }
  },
  "hints": [
    {
      "id": "5f7d...",
      "name": "Fantasy RPG",
      "summary": "A medieval fantasy RPG where a hero battles the demon lord."
    }
  ],
  "ui": {
    "theme": "dark",
    "language": "ja"
  }
}
```
